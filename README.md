My virtual pet - corgi Snickers.
Snickers is a very joyful and friendly dog, who loves to eat and not a secret that he has rather a chubby shape.

## Installation

Run the following commands in Terminal:
- $ git clone
- $ cd virtual-pet
- $ npm install
- $ npm run start
